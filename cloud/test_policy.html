Privacy Policy - CNR Biomics Cloud

The CNR Biomics Cloud service is committed to protecting user privacy in compliance with EU Regulation 2016/679 (GDPR) and adheres to the GÉANT Data Protection Code of Conduct.

1. Data Collection

This service receives the following minimum necessary attributes through LifeScience AAI for authentication and authorization purposes:

  • Unique User Identifier (Subject ID / eduperson_principal_name)
  • Email address
  • Display Name (First and Last Name)
  • Institutional affiliation and group memberships (entitlements)

2. Purpose of Processing

Personal data is processed exclusively to:

  • Provide access to the CNR Biomics Cloud computing resources.
  • Manage access permissions based on scientific group memberships.
  • Ensure system security and prevent unauthorized access or abuse.

3. Data Retention and Protection

Data is managed through a local Keycloak instance at CNR and is not shared with third parties. During this testing phase, access logs are kept for a maximum of 30 days for technical troubleshooting purposes.

4. Contact Information

For any questions regarding data management during this technical integration, please contact the technical administrator at the email address provided in the LifeScience AAI SP Registry.
